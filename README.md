<h1 style="text-align:center;">Kurumeko</h1>

## Recursos

* 🟨 JavaScript
* 🔥 Slash Commands
* 💪 Handlers/Eventos
* 🍃 MongoDB Suporte
* 🔪 Sistema de clã

## Instalação

Instalar os modulos
```
npm install
```

Crie um arquivo chamado `.env` e copie a informações abaixo

```js
TOKEN=SEUTOKEN
DatabaseToken=MONGOURL
```

Para iniciar o bot use `node .`
